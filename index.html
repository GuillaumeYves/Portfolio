Web RPG (WIP)

Mon dernier projet, qui est toujours en cours, est un web RPG type Shakes&Fidget.
Les fonctionnalités sont très similaires, je me suis permis d'ajouter un peu de complexité à travers les compétences et autres.
Ce projet me permet avant tout de pouvoir m'exercer avec une BDD plus volumineuse qu'un projet classique type "web".


- J'utilise la gem "devise" pour gérer ma connexion et création de compte
- La gem "whenever", pour les méthodes en background (toutes les X heures, do Y method)
- La BDD utilisée est PostgreSQL
- Ce projet est composé à 62.5% de Ruby, 35.9% de HTML et 1.6% de Javascript et autre
- Pour les illustrations, tout est fait grâce à une IA

Trouve ton Asso

Projet fait en groupe avec The Hacking Project.
Trouve ton Asso est une idée commune afin de rassembler les associations avec leurs bénévoles. La plateforme permet de s'inscrire et se connecter, faire un don ou encore rejoindre un évènement.
Il y a 3 type d'accès; L'user, L'association et L'admin. Ces 3 accès donnent l'autorisation à différentes fonctionnalités et pages du site.
Le projet est composé majoritairement de Ruby, ce qui nous permet d'utiliser des gems tels que stripe et devise. Nous avons aussi utilisé une api google maps sur les profils des associations, permettant des mieux les situer.
